三、代码仓提交方式
参赛选手使用fork，将仓库克隆到自己账户下开发，开发完成后申请PR，由评审确认符合要求后在PR里回复参赛选手，在比赛完后才会进行合并，对外公开。具体操作指南请查阅《附件1：AtomGit代码仓库使用指南》。

四、代码仓权限及硬件申请方式
下载并填写《附件2：参赛团队申请表》、《附件3：承诺函》，连同盖章版报名表（www.tiaozhanbei.net），以及硬件邮寄地址发送至 dianhong_jbgs@163.com，等待工作人员审核通过。邮件主题为：代码仓权限及硬件申请-申报人所在单位-申报人姓名-作品名称-联系电话（例如：代码仓权限及硬件申请-XX 大学-张 XX-XX 方案-手机号）
通过组委会材料审核后工作人员会根据《附件2：参赛团队申请表》填写的邮箱地址发送邀请链接，参赛者提交申请后由管理员审核，通过后成为“开发者”。
